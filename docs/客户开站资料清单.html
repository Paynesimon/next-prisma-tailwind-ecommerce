一、开站前确认

项目要求示例
客户 ID必填，全局唯一，英文小写client001
店铺名称必填TechGear Store
管理员邮箱必填，后台 OTP 登录admin@example.com
联系邮箱建议填展示在联系页
至少 1 个商品名称、价格、图片—
自定义域名选填，建站自动绑 Vercelshop.example.com

客户 ID 确定后勿改，用于数据库、GitHub 仓库、Vercel 项目命名。

二、店铺信息表

客户必填

字段说明
客户ID唯一标识 client003
店铺名称网站标题
店铺简介首页/SEO 描述
管理员邮箱后台登录

建议填写

关键词逗号分隔，SEO 用
联系邮箱 / 创建者客服与品牌名
自定义域名需客户能改 DNS

系统自动回写（勿手填）

前台/后台地址、部署状态、Sitemap、Google/Bing收录、SEO待办、DNS说明

每件商品一行，客户ID 须与店铺表一致

价格数字

品牌/分类/描述/关键词描述可 AI 生成

四、Banner 表（选填）

字段说明
客户ID必填（若做轮播）
图片URL / 标签名 / 排序数字越小越靠前

无 Banner 时用默认轮播。

五、支付配置表（选填）

字段说明
客户ID必填
Stripe 公钥/私钥pk_... / sk_...
PayPal Client ID若用 PayPal

不填可浏览，支付不可用。上线后需配 Stripe Webhook。

六、客户素材汇总 & 最小配置

  • 客户 ID（英文，唯一）
  • 店铺名称 + 简介
  • 管理员邮箱（能收验证码）
  • 商品清单（名称、价格、图片）
  • Logo / Banner 图（HTTPS）
  • 自定义域名（可选）
  • Stripe 密钥（若要收款）
  • 社交媒体链接（可选）

最小可建站（快速测试）

表最少字段
店铺表客户ID、店铺名称、店铺简介、管理员邮箱
商品表客户ID、商品名称、价格、图片URL（≥1条）

图片须 https:// 公网链接，勿用本地路径。改商品用 sync；改店名等需 build 重新部署。

七、建站流程

  1. 飞书填好店铺 + 商品（+ 支付，可选）
  2. 触发一键建站（action: build）
  3. 等待约 20–30 分钟
  4. 飞书自动回写前台/后台地址
  5. 有域名 → 按 DNS 说明解析
  6. 点「Google收录」提交 sitemap
  7. 日常改商品 → 自动 sync（约 1 分钟）
